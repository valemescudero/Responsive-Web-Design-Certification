Poncho

Poncho

The style and components library for the design of official websites and applications pertaining to the government of Argentina.

Why Use Poncho?

Unified experience

Citizens will recognize the website as part of the official Government's sites.

Accesible sites

Components have been optimized to garantee accesibility.

Save time and work

Copy and paste components and templates without having to code from scratch.

Tipography

Poncho's tipography is Encode.

Title and headings

There must be only one h1 title per page but there can be several headings. The h1 to h5 order must always be followed and no heading should be skipped.

Title Code

h1. This is a title

 <h1>Título</h1>

h2. This is a heading

 <h2>Heading</h2>

h3. This is a heading

 <h3>Heading</h3>

h4. This is a heading

 <h4>Heading</h4>
h5. This is a heading
 <h5>Heading</h5>

Text formating

These are the text elements that can be found in a text body and some interactive elements.

Format Code Function
Paragraph <p>Paragraph</p> General text
Bold <strong>Bold</strong> Make words stand out. Never entire paragraphs.
Italics <i>Italics</i> To indicate that a word or group of words have a special meaning
Small <small>Small</small> For texts with complementary information (eg. form fields indications, legal information, etc).

Lead

 <p class="lead">Lead</p> For standing out text that are not headings.
Quotes
 <blockquote>Quotes</blockquote> Use inside a different paragraph.

Text colors

Texts are usually black save for some interactive elements. They should not be used decoratively.
